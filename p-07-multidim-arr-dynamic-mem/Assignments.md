## Какво ще изведе кодът/каква е грешката?
### 1
```c++
int arr[5][5] = {{1,2,3,4,5},{1,2,3}};
std::cout << arr[0][3];
std::cout << arr[1][3];
std::cout << arr[3][3];
std::cout << arr[5][5];
```

### 2
```c++
int arr[3][3] = {1,2,3,4,5,6,7,8,9};
std::cout << arr[1][2];
```

### 3
```c++
int* p = new int(5);
std::cout << *p;

int& a = p;
std:: cout << a;
```

### 4
```c++
int* p = new int[5];
delete p;
```

### 5
```c++
int* p = new int(5);
delete[] p;
```

### 6
```c++
int* p1 = new int[10];
int* p2 = new int[5];
p1 = p2;

delete[] p1;
delete[] p2;
```

## Динамична памет

### Задача 5

```

```

